<color=silver>版本:1.2</color>

如何使用:
1.点击输入框输入你想通过识别匹配的文字
2.点击添加，也可通过删除，清空按钮管理列表
3.点击激活，即可使用当前相机拍照并将照片文字识别出来，并将于列表文字匹配，将识别和匹配结果显示出来，如果不匹配则会响起持续8秒的警报声
4.可点击相应按钮，浏览已录取信息所生成文本文档，和不匹配的图片，返回菜单及退出系统

<color=yellow>系统功能:</color>
<color=lime>1.主要功能
通过输入框录用文字信息，当被激活时，通过摄像头拍照，调用进行文字识别，发现与录取信息不匹配时即发出警报。
该功能主要用于模拟小区私人停车位的外来车牌识别与警报系统</color>
<color=orange>（1）输入并记录文字信息
（2）文字识别
（3）文字匹配
（4）警报系统</color>
 
<color=cyan>2.扩展功能
（1）对录取信息进行增加，删除与清空操作，并实时生成一个文本文档作为存档，在下次初始化应用时自动读取
（2）对不匹配的图片以时间格式保存，方便车主事后取证与查看记录</color>
  
<color=red>已知问题&注意事项:</color>
（1）由于unity原生截屏功能依赖像素坐标截取，目前仅适配<color=yellow>16:9</color>屏幕比例，该比例外的<color=red>拍照功能可能无法正确截取，且功能界面可能无法显示完全</color>
（2）改系统识图功能需要<color=yellow>全程联网</color>，<color=red>请保持网络畅通，否则容易造成死机现象</color>
